Detailed Description

A classe Register provê uma interface para facilitar a criação de Registros aceitos pelo banco de dados. Para isso, ela define um tipo propertyMap da seguinte maneira: typedef std::map<std::string, std::string> propertyMap;

Esse tipo é um map de colunas para valores, indicando qual o valor em cada coluna para o dado registro.
